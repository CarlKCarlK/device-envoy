A device abstraction for NeoPixel-style (WS2812) LED strips.

You can set each LED to an individual color.

This module treats a strip as a one-dimensional line of lights. If your LED strip forms a grid,
see Led2D, which supports text rendering, graphics, and animation.

In the first example, we define a strip of 48 LEDs and set every other LED to blue
